ParseError(input, expected, fatal)

extend()

Create an aggregate error by adding another error to the current one. If the new error occurs before the current in the buffer, it is simply ignored, and the result will be the current error. If it's at the same position, we return a new error containing the union of the two errors' expected messages. If it's ahead of the current error, we discard the current and return the new error unchanged.
